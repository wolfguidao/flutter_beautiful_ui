### ✨ 愿景与理念

欢迎来到我的创意空间！这个仓库不仅仅是代码的堆砌，它是对**美学**、**动效**和**用户体验**的深度探索。每一行代码都致力于挑战 Flutter 在视觉表现上的极限。

### 🚀 项目展示

以下展示了仓库中的独立实验项目。每个项目文件夹内都包含详细的文档说明。

####  [Awesome Interaction (酷炫交互)](./awesome_interaction)
专注于复杂手势、主题切换效果及微动效的实验场，展示 Flutter 强大的交互能力。

####  [CRM Workroom (CRM 工作台)](./crm_woorkroom)
拥有优美数据布局和流畅导航流程的专业级 CRM 管理后台界面。

####  [E-Robo Wallet (机器人钱包)](./E-Robo-Wallet)
未来主义风格的 AI 驱动 Web3 钱包，包含丰富的 3D 风格素材和现代化的加密货币管理 UI。

####  [SaaS Dashboard (SaaS 仪表盘)](./saas_dashboard)
极简主义、高可用的 SaaS 仪表盘，关注数据清晰度、性能表现和专业实用性。

> [!TIP]
> **进入文件夹探索！** 想要查看运行截图，请进入各项目目录并阅读其专门的 README 文档。

### 🛠️ 如何运行

只需几分钟，即可在您的本地设备上体验这些 UI：

1.  **克隆项目:**
    ```bash
    git clone https://github.com/JiuShuang/flutter_beautiful_ui.git
    cd flutter_beautiful_ui
    ```

2.  **进入指定项目:**
    ```bash
    cd E-Robo-Wallet  # 可替换为其他项目文件夹名称
    ```

3.  **同步与启动:**
    ```bash
    flutter pub get
    flutter run
    ```
